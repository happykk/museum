# 简介 - 下拉选择 - 支持多列 # 安装 ``` js import select from 'mk-ui/select.js' import 'mk-ui/select.css' import 'mk-ui/icon.css' /* 图标 */ Vue.use(select) ``` # 自定义 # 单列(demo) # 多列(demo) # mk-select (组件) ## 属性 |参数|说明|类型|可选值|默认值| |-|-|-|-|-| |v-model="selectedValue"|选中的值|string |type|类型|string|'primary'| ## 事件 |名称|说明| |-|-| |click|点击调用| |clear|点击右边'清除按钮'时调用| # mk-select-pop (组件) 底部浮层 ## 属性 |参数|说明|类型|可选值|默认值|返回值 |-|-|-|-|-|-| |v-model="selectedAr"|滚动区域选中的值|Array|||[{value: 1, text: 't1'}, ...] ## 事件 |名称|说明| |-|-| |shown|展示时调用| |cancel|取消时调用| |sure|确定时调用| # mk-select-pop-item (组件) 滚动层 ## options |参数|说明|类型|可选值|默认值 |-|-|-|-|-| |data|数据|Array|// 列表: text 表示展示文本, value 表示值
[{text: 't1', value: 1}, {text: 't2', value: 2}, {text: 't3', value: 3}] |onlyReturnValue|仅返回value的值(适用于text, value相等的情况, 如：时间组件，日历组件)
1. 若值为false, v-model="selectedAr1[0]" 返回{value:1, text:'t1'}
2. 若值为true, v-model="selectedAr1[0]" 返回1|boolean||false ## 属性 |参数|说明|类型|可选值|默认值 |-|-|-|-|-| |v-model="selectedAr1[0]"|选中的值(优先级大于default-value(属性))|string| |default-value|默认选中的值, 若没有则选中list的第一个值|string |title|标题|string |row|显示行数 (弹层的高) (必须为奇数: 3, 5, 7)|number||3| |position|文本显示位置|string||'center'| ## 事件 |名称|说明| |-|-| |scrollEnd|滚动结束时触发| |scrollBottom|滚动到底部时触发|
